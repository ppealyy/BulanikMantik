\# Bulanık Mantık Çamaşır Makinesi



Bu proje, C# WinForms kullanılarak geliştirilmiş bulanık mantık tabanlı

bir çamaşır makinesi simülasyonudur.



\## Girdiler

\- Hassaslık

\- Miktar

\- Kirlilik



\## Çıktılar

\- Dönüş Hızı

\- Süre

\- Deterjan
